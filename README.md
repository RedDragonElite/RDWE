## вҡЎ What is RDWE?

RDWE is a **PHP 8.4+ web engine** built from scratch on IIS + MariaDB with zero external framework dependencies.  
Drop in a module folder. Open a browser. Done. No registration. No config files. No restarts.

It's simultaneously a **CMS**, a **developer framework**, and a **community platform** вҖ” all in one zero-bloat stack.

## рҹ”Ҙ Feature Highlights

- **Auto-Discovery Module System** вҖ” create `Core/Classes/MyModule/UI/index.php` and it's live. Zero config.
- **PSR-4 Autoloader** вҖ” drop a class in the right folder, it loads itself
- **Singleton PDO Database Layer** вҖ” one connection per request, shared across all modules
- **Firewall with Admin Toggles** вҖ” CSP, X-Frame-Options, HSTS, rate limiting, IP blocking вҖ” all toggleable at runtime
- **RDUI Design System** вҖ” terminal aesthetic, monospace-only, red/green/black вҖ” consistent across every page
- **Elder Futhark Error Pages** вҖ” 403 (police tape), 404 (matrix rain), 500 (crash console + fire) вҖ” because style matters
- **Argon2ID + AES-256-GCM Auth** вҖ” OWASP 2024 compliant, session fingerprinting, dual rate limiting
- **Nostr Integration** вҖ” native NIP-07 signer Chrome extension + Nostr Terminal built-in
- **IIS-Native** вҖ” built for Windows Server / IIS + PHP 8.4 + MariaDB, not Apache-ported
- **BFS v6.66 License** вҖ” free forever, always

## рҹ—Ӯ Module System

Every module lives in `Core/Classes/ModuleName/` and gets auto-routed:

```
Core/Classes/MyModule/
в”ңв”Җв”Җ MyModule.php          вҶҗ extends Database
в”ңв”Җв”Җ MyModuleAPI.php       вҶҗ API dispatcher
в”ңв”Җв”Җ navdock.php           вҶҗ optional admin dock entry
в””в”Җв”Җ UI/
    в”ңв”Җв”Җ index.php         вҶ’ /MyModule
    в”ңв”Җв”Җ admin.php         вҶ’ /MyModule/Admin  (God-only)
    в””в”Җв”Җ api.php           вҶ’ /MyModule/API/{action}
```

**No route registration. No config files. No restart required.**  
URLRewriter scans `Core/Classes/` on every request and auto-discovers any folder with a `UI/` subfolder.
